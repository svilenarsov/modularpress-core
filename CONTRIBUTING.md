# Contributing to ModularPress

Thank you for your interest in ModularPress.

Before proceeding, please read this document carefully. ModularPress is **not an open source project** and contribution rules differ from typical public repositories.

---

## Commercial Product Notice

ModularPress is a **commercial CMS product** developed and maintained by **Network Technology**.

This repository does **not** contain the full product source code.
It serves as a public reference, documentation entry point and technical overview.

There is no implied open source license, community ownership or unrestricted contribution model.

---

## What This Repository Is For

This repository exists to:

- Provide high level documentation and technical references
- Explain architecture, modules and design philosophy
- Serve as a public point of reference for customers and partners
- Host non-sensitive assets such as previews and documentation

It is **not** intended to:
- Accept public feature implementations
- Accept core code changes
- Operate as a community driven codebase

---

## Contributions Policy

### Public Contributions

At this time, **external pull requests are not accepted** for:

- Core functionality
- Modules
- Architecture changes
- Security related logic
- Business logic or workflows

Unsolicited pull requests implementing features or changes will be closed without review.

This is intentional and aligned with the commercial nature of the product.

---

## Reporting Issues

If you have found:

- Documentation issues
- Typographical errors
- Broken links
- Inconsistencies in public descriptions

You may open an issue describing the problem clearly.

Please note:
- Issues are reviewed on a best effort basis
- Submitting an issue does not imply acceptance or prioritization

---

## Feature Requests and Custom Development

Feature requests, custom modules and integrations are handled **commercially**.

If you need:
- A new module
- Custom workflow
- Integration with internal systems
- Long term roadmap alignment

Please contact us through the official website:
https://modularpress.eu

---

## Security Issues

Security vulnerabilities must **not** be reported publicly.

If you believe you have discovered a security issue:
- Do not open a public issue
- Do not disclose details publicly

Contact us directly through the official channels listed on:
https://modularpress.eu/company
